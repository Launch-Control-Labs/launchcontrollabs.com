## Implementation Details

### Hook Logic Flow
```
useExperienceMode()
  ├─ useReducedMotion() → if true, return 'static'
  ├─ useIsMobile() → if true, return '2d-parallax'
  ├─ useDeviceTier() → if tier === 1, return '2d-parallax'
  └─ else → return '3d'
```

### Component Routing
```
<main data-experience-mode={mode}>
  ├─ mode === '3d' → <ScrollScene><HeroOverlay /></ScrollScene>
  ├─ mode === '2d-parallax' → <div>Mobile Experience — Coming in Task 14</div>
  └─ mode === 'static' → <div>Static Content — Coming in Task 17</div>
```
